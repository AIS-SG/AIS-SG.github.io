오프라인 한글 인식을 위한 전처리 알고리즘

본 프로젝트는 서경대학교 소프트웨어학과 김성석 교수님의 DI랩에서 수행되었으며, 오프라인 환경에서 한글 손글씨 인식을 가능하게 하기 위한 경량 전처리 알고리즘을 개발하고자 하였습니다. 대용량 데이터를 기반으로 하는 복잡한 인식 모델 대신, 모바일 환경에서도 동작 가능한 효율적인 사전처리 기반 구조를 설계했습니다.

🔍 프로젝트 개요

  • 목표: 자원 제약 환경에서도 사용할 수 있는 오프라인 한글 필기 인식 전처리 시스템 개발
  • 활용 도구: Python, OpenCV, NumPy
  • 적용 데이터: 필기 이미지 (KoHACC 데이터셋 기반), 직접 수집한 자필 이미지

🧠 주요 전처리 과정

1. 글자의 위치 조정

글자의 크기를 표준화하고, 이미지의 중앙에 정렬하여 위치 기반 왜곡을 제거합니다.

2. 세선화 (Thinning)

글자의 굵기를 1픽셀 두께로 줄여 획의 구조만 남깁니다. 이 과정에는 Zhang-Suen 알고리즘이 적용됩니다.

3. 점선으로 단순화

세선화된 선분을 점의 나열로 단순화하여 처리 연산량을 줄입니다.

4. 직선 획 구분

점선의 방향성과 근접성을 분석하여 자음·모음을 이루는 직선 획을 분리해냅니다.

5. 끊어진 획 연결 및 보정

획 간의 간격과 기울기를 기준으로 자연스럽게 이어질 수 있는 선분을 자동 연결해 자소 인식 정확도를 높입니다.

⚙️ 오프라인 인식 구조

전처리된 획 좌표는 기존 온라인 한글 인식 시스템에서 사용된 알고리즘과 호환되도록 설계되어 있습니다. 따라서 하나의 자모를 이루는 획 정보를 기반으로 초성/중성/종성을 구분하고, 이를 기반으로 자모 단위 인식이 가능합니다.

📉 기대 효과 및 성능

  • 모바일 및 저사양 환경에서도 높은 인식 성능 유지
  • 복잡한 딥러닝 없이 자모 분리만으로도 높은 인식 정확도 확보 가능
  • 전처리를 통해 CNN 모델의 입력 이미지 크기 축소 가능 → 연산량 감소
