아두이노를 이용한 미세먼지 센서

본 프로젝트는 서경대학교 소프트웨어학과 김성석 교수님의 DI랩에서 수행되었으며, 학교 운동장 및 교실의 미세먼지 데이터를 수집해 학교 맞춤형 미세먼지 경보 시스템을 구축하기 위한 기반 데이터 확보를 목적으로 진행되었습니다.

💡 초기 접근: 라즈베리파이 기반 센서

초기에는 Raspberry Pi Zero 2 W와 PMS7003 미세먼지 센서를 연결하여 Python 코드로 데이터를 수집하였으며, 30분 간격으로 측정된 데이터를 data_sheet.txt에 저장하고, 이후 PostgreSQL DB에 업로드하는 방식으로 설계되었습니다.

하지만 실외 환경에서의 전력 지속성 문제(예: 보조배터리로 약 2일만 유지)로 인해, Raspberry Pi 기반 센서는 장기 운영에 부적합하다는 판단이 있었습니다.

🔄 전환: 아두이노 기반 센서

이후 시스템은 전력 효율성과 절전 모드 제공이 가능한 Arduino UNO 기반으로 전환되었습니다. Arduino는 필요한 기능만 실행하며 불필요한 부하가 없어, Raspberry Pi보다 훨씬 낮은 소비 전력을 갖는다는 장점이 있습니다.

🔧 사용 부품

  • Arduino UNO
  • PMS7003 미세먼지 센서
  • SD 카드 리더 모듈
  • 이더넷 쉴드 (선택사항)

⚙️ 데이터 수집 및 저장 방식

센서를 통해 수집된 PM1.0, PM2.5, PM10 수치는 내부 타이머를 이용해 경과 시간과 함께 SD카드에 저장됩니다.

데이터는 아래와 같이 TEST.txt 파일에 저장됩니다:


34997, 9, 11, 12
61560, 11, 15, 15
62481, 10, 14, 16
63331, 10, 14, 16

이더넷 쉴드를 사용하는 경우, 수집된 데이터를 서버나 데이터베이스로 직접 전송하는 기능도 구현할 수 있습니다.

📸 센서 구성 요약

  • PMS7003 센서 → 아두이노 시리얼 통신으로 연결
  • SD 리더기 또는 이더넷 쉴드로 데이터 저장
  • 라즈베리파이보다 낮은 전력 소모로 외부 사용에 적합
